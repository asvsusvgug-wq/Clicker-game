# 💰 Кликер - Заработай Миллионы

Простой, но увлекательный клicker-game с прокачками, магазином и системой перерождений!

## 🎮 Возможности

- **🖱️ Кликер** - Основной механизм: кликайте кнопку и зарабатывайте деньги
- **🏪 Магазин** - Покупайте улучшения:
  - 🖱️ Мышь (+1 доход/клик)
  - ⌨️ Клавиатура (+2 доход/клик)
  - 🖥️ Монитор (+5 доход/клик)

- **⬆️ Активные Прокачки**:
  - 2️⃣ Двойной Клик - каждый клик считается дважды
  - ✖️ Множитель - все клики × 2
  - 🤖 Автокликер - +1 клик в сек автоматически

- **🔄 Система Перерождений**:
  - Получайте звезды за достижения
  - +5% к доходу за каждую звезду
  - Сбросьте прогресс, чтобы начать заново с бонусами

- **📊 Статистика**:
  - Отслеживание всех кликов
  - Общий заработок
  - Время игры
  - Средний доход в секунду

- **💾 Сохранение**:
  - Сохраняйте прогресс в localStorage
  - Загружайте игру в любой момент

## 🚀 Быстрый Старт

1. Откройте `index.html` в браузере
2. Начните кликать! 🎯
3. Зарабатывайте деньги и покупайте улучшения
4. Открывайте новые прокачки
5. Используйте систему перерождений для ускорения прогресса

## 🔧 Как Модифицировать

### Добавить новый предмет в магазин:
```javascript
{ 
    id: 'gpu', 
    name: '🎮 Видеокарта', 
    basePrice: 2000, 
    description: '+10 доход/клик', 
    level: 0, 
    effect: () => gameState.clicksPerSecond += 10 
}
```

### Добавить новую прокачку:
```javascript
{ 
    id: 'megaClick', 
    name: '💥 Мега-клик', 
    price: 50000, 
    description: 'Каждый клик × 5!', 
    purchased: false 
}
```

## 💡 Советы Для Игроков

1. **Начните с мыши** - дешёвое первое улучшение
2. **Сохраняйте часто** - используйте кнопку "Сохранить"
3. **Изучайте активные прокачки** - они дают огромные бонусы
4. **Перерождайтесь при наличии звёзд** - экспоненциальный рост силы
5. **Оптимизируйте** - найдите баланс между кликами и автодоходом
